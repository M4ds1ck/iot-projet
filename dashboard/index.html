CHEMCHAM — DEVICE INTERNALS & SPECIFICATIONS
CHEMCHAM — IoT RF Monitor v1.0

A fully passive, wide-band RF scanning device built on a Raspberry Pi Zero 2W and RTL-SDR v3 dongle. CHEMCHAM does not transmit — it only listens. It detects signals across 7 frequency bands, attributes them to registered field devices by frequency matching, and streams analysis to this dashboard in real time over WebSocket.

Components
🍓
Raspberry Pi Zero 2W
Compute core. Runs Python scanner daemon, connects to WiFi, streams detections via MQTT → WebSocket. Quad-core ARM, 512MB RAM.
🔌
RTL-SDR v3 Dongle
SDR front-end using RTL2832U + R820T2. Converts RF → 8-bit IQ samples at up to 3.2 MS/s. Connected via USB OTG. Direct sampling mode covers 24 MHz (HF).
📶
Telescopic Whip Antenna (SMA)
Quarter-wave. Extend to 16.4 cm for PMR446, 17.3 cm for ISM-433. Replaceable with bandpass antenna for single-band selectivity.
